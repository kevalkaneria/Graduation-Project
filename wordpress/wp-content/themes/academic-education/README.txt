== Steps to Setup Theme ==

Below are the following step to setup theme static page.
=========================================================
Step 1. Add new page named as "home page" and select the template "Static Front Page".
Step 2. Go to customizer >> static front page >> check Static page, then select the page which you have added example "home page".

For SLider
==============
Step 1. Add new page, add title, content and featured image and then publish it.
Step 2. Go to customizer >> Theme Settings >> Slider settings >> here you can select the page which you have add for slider.

For Social Link
====================
Step 1. Go to customizer >> Theme Settings >> Social Media >> here you can add your socail links.

For Topbar
==============
Step 1. Go to customizer >> Theme Settings >> Topbar >> here you can add your Contact details.

For Courses Section 
=========================
	For Post:
	Step 1. Add new post, add title, content and featured image and publish it.
	Step 2. Go to customizer >> Theme Settings >> Courses Section >> here you will get option to "select post", select your post and publish it.

	For Category:
	Step 1. Add new Category.
	Step 2. Add new post, add title, content, assign category, featured image and publish it.
	Step 3. Go to customizer >> Theme Settings >> Courses Section >> here you will get option to "Select Category", select your Category and publish it.

For General Settings
======================
Step 1. Go to customizer >> Theme Settings >> General Settings >> here you can change the layout of the blog post.
